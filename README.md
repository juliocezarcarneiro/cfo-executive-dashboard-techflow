## Project Overview

This project delivers a CFO-ready financial dashboard designed to replace static reporting with an interactive, insight-driven reporting experience.

The dashboard is structured around four business questions:

- How is overall revenue performing?
- Are profit trends keeping pace with sales growth?
- Where are we underperforming against budget?
- What is happening across departments and regions?

## Dashboard Scope

The final dashboard includes the following analysis areas:

1. Executive Summary  
2. Revenue & Profit Trends  
3. Budget vs Actual Variance  
4. Departmental & Regional Breakdown  

## Key Insights

- Revenue reached **$110.2M**
- Net profit reached **$52.2M**
- Profit performance finished **10.1% below budget**
- Margin pressure suggests rising efficiency challenges
- Budget underperformance was concentrated in **Sales and Engineering**
- Cash flow remained stable despite profit pressure
